# OpenSpec Specifications

`openspec/specs/*` хранит канонические capability/process specifications для `/int/tools`.

Для governance tracked tooling mutations базовый source-of-truth:

- `openspec/specs/process/spec.md`

Новые capability specs создаются только по явному owner approval; по умолчанию существующие specs расширяются, а не дублируются.
